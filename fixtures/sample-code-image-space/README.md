# Fixture: sample-code-image-space

Cette fixture représente un petit Space Layrs lisible sans dépendances. Elle mélange un exemple de code textuel, une représentation d'image en ASCII et quelques objets produit simulés.

Elle sert à tester les futurs parseurs, stores et vues sans binaires lourds.

## Contenu

- `space.layrs.txt`: métadonnées humaines du Space.
- `src/hello.layrs.txt`: pseudo-code simple.
- `artifacts/ui-frame.txt`: représentation textuelle d'un artifact visuel.
- `layers/base.txt`: état de base.
- `layers/experiment-copy.txt`: Layer expérimentale.
- `steps/render-preview.txt`: Step simulé.
- `weaves/main-thread.txt`: fil narratif minimal.
- `notes/review.txt`: note de revue.
